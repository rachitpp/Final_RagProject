# Leave Without Pay (LWOP) — payment of weekly-offs (HRP/018, §6.6)

Clean transcription of the scenario table on page 4 of the leave policy (the
source PDF draws it without ruling lines, so it cannot be auto-extracted).
Verbatim from leave.pdf, p.4.

## How to read this table (READ FIRST)
Find the row whose Friday and Monday status match the question. Its **Outcome**
column lists EXACTLY the days that are NOT paid. Apply this rule mechanically:
**a day is unpaid if and only if the matching row's Outcome names it; every day
NOT named is PAID** (this includes the Saturday/Sunday weekly-offs). This
scenario table is the policy's authoritative worked-out result for these cases —
use the matching row's Outcome directly and do not re-derive a different answer
from any general wording elsewhere.

(Saturday and Sunday are always the weekly-offs in question. A blank Tuesday is
an ordinary working day not relevant to that scenario.)

| Friday | Saturday | Sunday | Monday | Tuesday | Outcome — days NOT paid |
| --- | --- | --- | --- | --- | --- |
| Present | weekly-off | weekly-off | Present |  | (none — all paid) |
| LWOP | weekly-off | weekly-off | LWOP |  | Fri, Sat, Sun, Mon |
| LWOP | weekly-off | weekly-off | Present |  | Fri, Sat, Sun |
| Present | weekly-off | weekly-off | LWOP |  | Mon |
| Present | weekly-off | weekly-off | Holiday | Present | (none — Sat, Sun & Mon paid) |
| LWOP | weekly-off | weekly-off | Holiday | LWOP | Fri, Sat, Sun, Mon, Tue |
| LWOP | weekly-off | weekly-off | Holiday | Present | Fri, Sat, Sun, Mon |
| Present | weekly-off | weekly-off | Holiday | LWOP | Tue |

Worked example of the rule (both directions, so the rule is applied symmetrically):
- Row **Present Friday, LWOP Monday** → Outcome names only "Mon" → Monday is
  unpaid; Saturday & Sunday are NOT named, so they are **paid**.
- Row **LWOP Friday, Present Monday** → Outcome names "Fri, Sat, Sun" → all three
  are unpaid, so Saturday & Sunday are **not paid** here.

(Background, §6.3: holidays or weekly-offs that fall *during* a run of LWOP are
themselves counted as LWOP days. Misuse to gain paid weekly-offs may be refused
per the policy's note.)
